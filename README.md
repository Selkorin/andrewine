# ANDREWINE — перенос Tilda в Astro без редизайна

Проект сохраняет исходный дизайн, адаптивную вёрстку, тексты и все тематические страницы из выгрузки Tilda.

## Маршруты

- `/`
- `/buy_champagne`
- `/buy_whisky`
- `/buy_cognac`
- `/buy_portwine`
- `/buy_rum`
- `/buy_vodka`
- `/buy_brandy`
- `/buy_wine`

## Запуск

```bash
npm install
npm run dev
```

Проверка и сборка:

```bash
npm run build
```

Оригинальные HTML-страницы, включая три неопубликованных черновика, лежат в `src/legacy`. Внешний вид можно улучшать поэтапно, не переписывая сайт целиком.
